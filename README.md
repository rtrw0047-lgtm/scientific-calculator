# Scientific Calculator

Scientific calculator with trigonometry, logarithms, powers, and history.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Calculator** (or Skip).

**Step 3:** Click number buttons or type on your keyboard to enter values.

**Step 4:** Use operators (+, -, *, /) between numbers. Click = to evaluate.

**Step 5:** Use scientific functions:
- **sin, cos, tan** - trigonometry (radians)
- **log** - base-10 logarithm
- **ln** - natural logarithm
- **sqrt** - square root
- **x^y** - power
- **x²** - square
- **1/x** - reciprocal
- **|x|** - absolute value
- **π** and **e** - constants

**Step 6:** Past calculations appear in the history. Click any to reload it.

**Step 7:** Keyboard support: numbers, operators, Enter (=), Backspace, Escape (clear).

## Features
- Basic arithmetic (+, -, *, /)
- Trigonometry (sin, cos, tan)
- Logarithms (log, ln)
- Powers and roots
- Constants (π, e)
- Calculation history
- Keyboard support
- Parentheses support
